• Uri 1001:

- Split: divide uma string em substrings.

- parseInt(): converte uma string para número.

- shift(): remove o primeiro elemento(0) e retorna o valor removido.

• Uri 1002:

- parseFloat(): retorna um número de ponto flutuante.

- Math.pow(): retorna a base elevada ao expoente. 

- toFixed(): arredonda para cima, com ou sem casas decimais. 

• Uri 1012

- map(): percorre um array e modifica cada elemento desse array. retorna um array com a mesma quantidade de elementos do array original, mas com cada um desses elementos modificados.

• Uri 1013

- Math.abs(x): retorna o valor absoluto de um número "x". 

• Uri 1015

- Math.sqrt(): retorna a raiz quadrada de um número.
